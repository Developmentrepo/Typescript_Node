# Demo Service

This service is a Microservice that runs in Docker Containers. Stack Used:

#TypeScript
#Docker
#PostgreSQL
#RabbitMQ
#Microservices

## Installation

Use the npm ecosystem for packages

```bash
npm install
```

## Usage

Compile using
```bash
npm run compile
```

Compile with Monitoring using
```bash
npm run start:watch
```
